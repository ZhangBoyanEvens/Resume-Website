# Zhang Boyan 简历作品集

一个现代化的个人简历作品集网站，使用 React + TypeScript + Tailwind CSS 构建。

## 功能特性

- 现代化的响应式设计
- 基于 Vite 的快速开发体验
- 移动端友好
- 流畅的动画效果
- TypeScript 支持
- Tailwind CSS 样式系统

## 技术栈

- **前端框架**: React 18
- **开发语言**: TypeScript
- **构建工具**: Vite
- **样式框架**: Tailwind CSS
- **路由**: React Router DOM
- **动画**: Framer Motion
- **图标**: Lucide React
